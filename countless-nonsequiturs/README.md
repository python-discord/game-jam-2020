### RUNME

Windows CMD:
```cmd
$ python -m venv venv
$ venv\Scripts\activate
(venv) $ pip install -r requirements.txt
(venv) $ python -m client
```

Otherwise:
```sh
$ python -m venv venv/
$ source venv/Scripts/activate
(venv) $ pip install -r requirements.txt
(venv) $ python -m client
```

Exit the virtualenv with:
```sh
(venv) $ deactivate
```
